```shell
npm i sugarshin/smoothscroll
```

## Usage

```coffeescript
SmoothScroll = require 'smoothscroll'

new SmoothScroll element
```

or

```html
<script src="jquery.js"></script>
<script src="smoothscroll.js"></script>
<script>
  new SmoothScroll(element);
</script>
```
